### Windows 11 IPv6 配置

**问题**：无法获得 IPv6 地址或路由

**快速修复**：

```cmd
# 查看接口索引
ipconfig /all

# 删除旧路由（替换 15 为实际接口索引）
netsh interface ipv6 delete route ::/0 interface=15

# 启用路由器发现
netsh interface ipv6 set interface 15 routerdiscovery=enabled

# 拔插网线或重连 Wi-Fi 刷新 RA
```
